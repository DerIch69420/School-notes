Hyperbel

Eine Hyperbel ist eine ist eine starke Übertreibung, die in der Rethorik verwendet wird, um etwas besonders hervorzuheben oder eine starke Wirkung zu erzielen. Sie wird genutzt, um eine Aussage zu verstärken oder zu dramatisieren, indem etwas weit über das normale Maß hinaus beschrieben wird.

Beispiele für Hyperbeln:

  • "Ich habe dir das schon tausendmal gesagt!" (Hier wird die Zahl "tausend" übertrieben verwendet, um auszudrücken, dass etwas sehr oft gesagt wurde.)
  • "Das dauert eine Ewigkeit!" (In Wirklichkeit dauert es nicht wirklich eine Ewigkeit, aber es wird als besonders lang empfunden.)
